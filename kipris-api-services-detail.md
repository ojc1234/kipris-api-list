### 1. 특허·실용 공개·등록공보

- **서비스ID:** `DBII_000000000000001`
- **분류체계:** 국내 IP데이터 > 공보 > 특허·실용
- **제공기관:** 지식재산처
- **업데이트주기:** 일 단위
- **데이터 등록일:** 2014-10-20
- **데이터 갱신일:** 2020-10-27
- **태그:** 등록공보, 공보, 특허, 실용신안, 공개공보, 실용, 특허실용 공개등록공보, 공개공보
- **서비스유형:** REST API

#### 개요
한국 특허 / 실용신안 공개 및 등록공보의 **서지정보, 대표도, 전문정보** 등을 API(REST방식)으로 제공

#### 세부 정보 내용
1. **검색** — 일반검색 및 항목별검색 기능 통해 특허·실용신안 공보 정보를 XML 형태로 제공
2. **서지정보** — 출원번호, 등록일자, 발명의 명칭, 등록상태 등의 정보 XML 형태로 제공
3. **도면/전문** — 공개 및 공고 전문, 대표도면의 다운로드 경로 XML 형태로 제공
